# Таблица

## Основная задача программы

Фильтрация таблицы по категориям

## Платформа программы

Используется современный подход создании программы с помощью библиотеки React(для создания пользовательских интерфейсов)
Основным компонентом является index.js, репозитории разбит на мелкие компоненты (Header, Main, Footer, Pagination, Filtering....).
Применен стилевой подход scss. Использован json-server(подключены зависемости: concurrently, nodemon).

## Запуск программы

Перед тем как запустить проект на своем компьютере необходимо запустить команду npm i, для совместноного запуска json-server 
написать команду npm run dev.
